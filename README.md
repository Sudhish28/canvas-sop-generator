# AI-Powered Canvas SOP Generator & Workflow Analyzer

## 📌 Project Overview

This project is an AI-powered prototype that converts unstructured Canvas workflow descriptions into structured Standard Operating Procedures (SOPs), process diagrams, and workflow analysis insights.

The system focuses on common student activities such as:
- Logging into Canvas
- Navigating courses
- Accessing modules
- Submitting assignments

The goal is to improve clarity, reduce confusion, and identify inefficiencies in typical student workflows.

---

## 🎯 Current Features (Prototype Stage)

✔ SOP generation from workflow text input  
✔ Basic workflow structuring into steps  
✔ Mermaid diagram generation (process visualization)  
✔ Simple workflow analysis (bottlenecks, redundancies, suggestions)  
✔ Sample execution script for demonstration  

---

## 🧠 System Modules

### 1. SOP Generator
Converts user workflow descriptions into structured SOP format including:
- Purpose
- Step-by-step instructions
- Input/Output description

### 2. Diagram Generator
Creates a basic Mermaid flowchart representation of the workflow.

### 3. Workflow Analyzer
Identifies:
- Potential bottlenecks  
- Redundant steps  
- Improvement suggestions
