# AI-Powered Canvas SOP Generator & Workflow Analyzer

This project generates Standard Operating Procedures (SOPs) from Canvas workflow descriptions using AI.

## Features (in progress)
- SOP generation from text input
- Workflow diagram generation (Mermaid)
- Workflow analysis (bottlenecks + improvements)

## Tech Stack
- Python
- AI API (LLM-based)
- Mermaid diagrams

## Status
Early development stage (checkpoint submission in progress)
